Goal

The Bright Side Remoting component goal is to transparently encapsulate remote calls on EJB in HTTP.

The main use of it is for rich client startable by webstart. The solution of a dynamic deployment associated with a firewall friendly protocol leads to a very useful architecture.

We use this solution for all our component demonstration so go on The Bright Side Factory Community Site to see this component in action.
